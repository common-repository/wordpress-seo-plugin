== Installation ==
= To Install: =

1.  Download the WPMU DEV SEO plugin file
2.  Unzip the file into a folder on your hard drive
3.  Upload the `/wordpress-seo/` folder to the `/wp-content/plugins/` folder on your site
4.  Visit Network Admin -> Plugins and Network Activate it there.
5.  Alternatively, you can activate the plugin on individual sites by going to the Site Admin -> Plugins menu and activating there.
